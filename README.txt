A Pen created with CodePen.io, exported & developed locally then
pushed to GitHub pages

You can find this one at https://adventurebear.github.io/rhino-crunch/

User Stories:
1) I can upload a custom CSV file from my laptop that will display as HTML
2) I can input new data for two input fields and the data will be bound to a
   locally stored object
3) I can score my entries with a custom scoring system
4) I can download the sorted data as a CSV file to my laptop
